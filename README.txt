Mr White – minimál projekt
==========================

Futtatás:
  npm install
  npm run dev
  -> http://localhost:5173/#home

Build:
  npm run build
  -> dist/ mappát töltsd fel a tárhelyre

Képek a public/ mappában:
  aa.svg      (logó)
  nav.webp    (navbar háttér)
  og.jpg      (OG kép megosztáshoz)
  (SEO csomag fájlok: favicon.ico, apple-touch-icon.png, robots.txt, sitemap.xml, manifest.webmanifest)

Forráskód:
  src/MrWhiteSite.tsx   (a teljes oldal)
  src/main.tsx          (indító)
